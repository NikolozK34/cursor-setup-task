## Issues Faced & Solutions

### Issue 1: Extension login
Problem: Login didn’t work initially  
Solution: Restarted Cursor and logged in again

### Issue 2: Git not detected
Problem: Cursor didn’t detect Git  
Solution: Installed Git and restarted Cursor

### Issue 3: Push failed
Problem: Couldn’t push changes  
Solution: Logged into GitHub via terminal using:
git config --global user.name "YourName"
git config --global user.email "your@email.com"
